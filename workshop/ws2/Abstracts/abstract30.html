Paper 30
Title:Towards Semantic Interoperability in WoT Ecosystems
Paper:(May 31, 15:23 GMT)(previous versions)
Author keywords:
Semantic interoperability
WoT discovery
Heterogeneous data
RDF virtualisation
Discovery and distributed access
EasyChair keyphrases:gateway api (206), iot infrastructure (140), contextual data (90), thing description (80), distributed query client (63), sparql query (60), semantic repository (60), relevant data (50), node gateway api (47), gateway api service (47), captured data (40)
Abstract:IoT infrastructures publishing their data on the Web have become pervasive. Nevertheless, data is published under heterogeneous formats and models. In this paper we introduce an interoperability approach based on the WoT ontology, that allows to define contextual data for a given IoT infrastructure, define such infrastructure relaying on the WoT ontology, and fetch its data and translate it into RDF on the fly. Our approach has been implemented in the European h2020 project VICINITY. The main benefits of our proposal are the transparent discovery and distributed access capabilities.
