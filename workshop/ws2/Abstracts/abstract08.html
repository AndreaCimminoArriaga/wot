Title:Scripting API for the Web of Things

Abstract:The WoT Scripting API is an optional "convenience" building block in WoT and it is typically used in gateways that are able to run a WoT Runtime. It provides a convenient way to work with a Thing Descriptions and WoT Interactions (Properties, Actions and Events defined by a Thing). It allows extending WoT support to new types of endpoints and implement WoT applications such as Thing Directory. It allows remotely representing and also extending an existing Thing functionality. It allows creating, extending and composing services that use WoT Interactions.

The API follows ECMAScript syntax but it is not bound to it. Similar APIs can be implemented in multiple languages and runtimes.
